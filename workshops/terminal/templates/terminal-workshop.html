Prerequisites

  • Attend "Introduction to the Unix Terminal"
  • If you are using a Windows computer, install PuTTY

Instructions

Part 1

  1. List your current directory
  2. Print the location of your working directory
  3. Create a new directory called 'hello'
  4. Enter the 'hello' directory

Part 2

  1. Create a file 'about-me.txt'
  2. Add your name to 'about-me.txt'
  3. Append a fun fact about you (on a newline) in 'about-me.txt'
  4. Print the contents of 'about-me.txt'
  5. Type the command $ curl -F data="@about-me.txt" {{ domain }}
